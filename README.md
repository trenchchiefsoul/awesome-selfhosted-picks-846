## Productivity

Notes, knowledge bases, kanban, and personal workspace tools.

- [Affine](https://github.com/trenchchiefsoul/awesome-selfhosted-picks-846) — Local-first Notion and Miro alternative.
- [Colanode](https://github.com/colanode/colanode) — Local-first Slack and Notion alternative.
- [Docmost](https://github.com/docmost/docmost) — Open-source Notion and Confluence alternative.
- [Excalidraw](https://github.com/excalidraw/excalidraw) — Virtual whiteboard for hand-drawn-style diagrams.
- [Kanba](https://github.com/Kanba-co/kanba) — Minimal open-source Trello alternative.
- [Reactive Resume](https://github.com/AmruthPillai/Reactive-Resume) — Free and open-source resume builder.
- [Slash](https://github.com/yourselfhosted/slash) — Open-source URL shortener and bookmarks manager.

## Communication & Email

Customer support, email infrastructure, newsletters, and notifications.

- [Chatwoot](https://github.com/chatwoot/chatwoot) — Open-source customer support and Intercom alternative.
- [Gmail Cleaner](https://github.com/Gururagavendra/gmail-cleaner) — Bulk unsubscribe and clean Gmail locally.
- [Listmonk](https://github.com/knadh/listmonk) — High-performance self-hosted newsletter and mailing list manager.
- [Notifuse](https://github.com/Notifuse/notifuse) — Self-hosted Mailchimp and Brevo alternative.
- [Novu](https://github.com/novuhq/novu) — Open-source multi-channel notifications infrastructure.
- [Open Archiver](https://github.com/LogicLabs-OU/OpenArchiver) — Self-hosted email archiving and search.
- [Sessy](https://github.com/marckohlbrugge/sessy) — Open-source email observability for AWS SES.
- [useSend](https://github.com/usesend/useSend) — Open-source Resend and SendGrid alternative.

## Storage & File Sharing

Self-hosted cloud storage and peer-to-peer file transfer.

- [Cloudreve](https://github.com/cloudreve/Cloudreve) — Self-hosted cloud storage with multi-cloud support.
- [FileRise](https://github.com/error311/FileRise) — Self-hosted file manager with WebDAV.
- [LocalSend](https://github.com/localsend/localsend) — AirDrop-style file sharing across all platforms.
- [ownCloud](https://github.com/owncloud/core) — Self-hosted file sync and content collaboration.
- [OxiCloud](https://github.com/DioCrafts/OxiCloud) — Ultra-fast self-hosted cloud built in Rust.
- [Transfer.zip](https://github.com/robinkarlberg/transfer.zip-web) — Self-hosted WeTransfer alternative for file sharing.
- [Zipline](https://github.com/diced/zipline) — Powerful self-hosted file and image uploader.

## Media

Video editing, photo galleries, image optimization, and screen recording.

- [Cap (screen recording)](https://github.com/CapSoftware/Cap) — Open-source Loom alternative for screen recordings.
- [Kimu](https://github.com/trykimu/videoeditor) — AI-powered browser-based video editor.
- [Kriti Images](https://github.com/kritihq/kriti-images) — Open-source Cloudflare Images alternative.
- [Mazanoke](https://github.com/civilblur/mazanoke) — In-browser image optimizer that works offline.
- [Photofield](https://github.com/SmilyOrg/photofield) — Blazing-fast self-hosted photo gallery viewer.

## Documents

PDF management and document signing.

- [Documenso](https://github.com/documenso/documenso) — Open-source DocuSign alternative for documents.
- [DocuSeal](https://github.com/docusealco/docuseal) — Open-source DocuSign alternative for signing.
- [PDFDing](https://github.com/mrmn2/PdfDing) — Self-hosted PDF manager, viewer, and editor.

## Analytics

Privacy-friendly web analytics and telemetry.

- [Ackee](https://github.com/electerious/Ackee) — Self-hosted, node-based privacy analytics.
- [Rybbit](https://github.com/rybbit-io/rybbit) — Privacy-friendly Google Analytics alternative.
- [Self-hosted Metrics](https://self-hosted-metrics.com) — Privacy-first telemetry for self-hosted apps.
- [Umami](https://github.com/umami-software/umami) — Simple, privacy-focused web analytics.

- [Passbolt](https://github.com/passbolt/passbolt_api) — Open-source password manager for teams.
